# Suggested Commands

## Development
- `npx tsx src/cli.ts [command]` - Run CLI in dev mode
- `npx tsc --noEmit` - Type check without emitting
- `npm run build` - Production build

## Testing
- `npx tsx test_scripts/test-registry.ts` - Registry tests (14)
- `npx tsx test_scripts/test-git.ts` - Git module tests (10)
- `npx tsx test_scripts/test-cli.ts` - CLI integration tests (14)
- `npx tsx test_scripts/test-ai-config.ts` - AI config tests (10)
- `npx tsx test_scripts/test-repo-content.ts` - Content collector tests (9)
- `npx tsx test_scripts/test-describe-cli.ts` - Describe CLI tests (5)

## Install
- `npm link` - Global install after build

## System Utils
- `git`, `ls`, `cd`, `grep` (standard macOS/Darwin)
